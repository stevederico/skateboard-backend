# starter-backend
a basic backend service for user authenication and sqlite.

## Features
- Deno 2.2
- SQLite
- REST API for User Authenication

## Getting Started
```shell
deno run --allow-net --allow-write --allow-read index.js
```
## Usage
```shell
deno run start
```
